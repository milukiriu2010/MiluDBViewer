1. Introduction

MiluDBViewer is a free database GUI Tool.

2. Install & Run

2-1. Windows
(1) run "MiluDBViewer_Setup*.*.*.exe"
(2) run "<<your path>>\MiluDBViewer.exe"

2-2. Linux
(1) extract MiluDBViewer*.*.*.tgz by a file archiver to <<your path>>
(2) $ cd <<your path>>
(3) $ ./MiluDBViewer***.sh
