1. Introduction

MiluDBViewer is a free GUI DB Tool.

2. Install

2-1. Windows
(1) extract MiluDBViewer***.zip by a file archiver to <<your path>>
(2) cd <<your path>>
(3) click MiluDBViewer***.bat

2-2. Linux
(1) extract MiluDBViewer***.zip by a file archiver to <<your path>>
(2) $ cd <<your path>>
(3) $ ./MiluDBViewer***.sh
