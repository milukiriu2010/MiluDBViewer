3. Requirements
============================================================
JRE/JDK 10.0.1 or later
+ JavaFX

**************************************************
* JavaFX                                         *
**************************************************
JavaFX is not included since jdk11.
https://openjfx.io/openjfx-docs/#install-java
https://openjfx.io/openjfx-docs/#install-javafx

necessary to set javafx library path
like
PATH_TO_FX=C:\Program Files\Java\javafx-sdk-11\lib

============================================================
4. Install & Run
============================================================
4-1. Windows
(1) run "MiluDBViewer_Setup*.*.*.exe"
(2) run "<<your path>>\MiluDBViewer.exe"

4-2. Linux & Mac
(1) extract MiluDBViewer*.*.*.tgz by a file archiver to <<your path>>
(2) $ cd <<your path>>
(3) $ ./MiluDBViewer***.sh


============================================================
5. Uninstall
============================================================
5-1. Windows
(1) go to [Control Panel]-[Programs and Features]
(2) select "MiluDBViewer" and uninstall

5-2. Linux & Mac
(1) rm -rf <<your path>>
